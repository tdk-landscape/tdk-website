Features

Everything you need for local microservice development

Without the Kubernetes complexity.

Manifest-Driven

One service.json is the single source of truth. All configs generated from it.

Golden Images

Production-ready Docker layering with L1 (OS) → L2 (Runtime) → L3 (Deps) → L4 (Code).

Auto-Discovery

Automatically finds all service.json files in your directory structure.

Hot Reload

Built on Tilt with automatic reloading. See changes instantly as you code.

10+ Config Generators

Docker, Vite, TypeScript, Nginx, Environment files, and more—all automatic.

Proven Scale

Tested with 120+ services running locally with selective startup.
